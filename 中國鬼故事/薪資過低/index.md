# 中國鬼故事/薪資過低

## 中國/法律系/律師/強制社保/經濟不景氣/就業困難

```
有位中國法律系女畢業生，現在實習律師的生活讓人心疼
——她每月薪水是 2500 元人民幣，💸 扣了高達 1800 元的社保費，結帳後只剩下 700 元。
平均每天工作下來，她是拿人錢還倒貼上班，真的快撐不下去了！

她解釋說，這是「規定」——如果不繳社保，就沒辦法拿到實習律師證進而成為律師，工作機會根本被鎖死。🚇 這位女生通勤一天就要花 12 元人民幣，還得吃飯、租房，根本付不起自己上班的開銷。她淚訴：「靠家教和長輩資助支撐，否則我真的活不下去。」

這段經歷在網路上引起共鳴，大家心疼又感同身受。有留言說：「堅持就撐過去了，但也早點換條路吧。」有的則酸：「我們都像牛馬一樣被用過。」「不要學法律，畢業就要做技術員。」

這也是反映出現今中國就業情況的縮影：經濟下行、大學生找不到好工作、勉強撐著上班，卻換來的是更深的焦慮與無力。
```

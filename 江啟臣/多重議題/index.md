# 江啟臣/多重議題

## 江啟臣/發展停滯/交通/治安/霸凌/罷免/台中

```
你還記得嗎？
曾幾何時，我們以台中為傲——那是陽光灑落的城市。
但現在？這座城市變得我們越來越不認得了。

台中再度成為「行人地獄」的第一名，
下個雨，街道就變成「漂漂河」，
受夠了市政府帶頭賣地炒房，讓年輕人看不見未來，
受夠了跨年煙火從歡慶變成「落焰戰場」，讓人心驚膽跳。

韓團來台被吃豆腐的羞辱，台中竟成為「詐騙博弈的首都」，
孩子被學校霸凌、被傷害，
而領政的市府卻選擇轉頭、沉默、冷漠。
這些，早已不只是新聞。
而是我們每日生活的真實、每日累積的失望。

讓我們說出心中的怒吼——
我們要真正改變，不能再讓國民黨與三寶立委繼續擺爛台中！
```

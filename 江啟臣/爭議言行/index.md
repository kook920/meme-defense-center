# 江啟臣/爭議言行

## 江啟臣/國民黨/干擾/台中/罷免

```
你們聲音那麼大，是因為心裡有底氣，還是心虛？

今天下午，我們的志工照常站上南陽路路口，不為別的，只為讓更多人關心家鄉的未來。
我們沒什麼資源，也沒有政黨撐腰，只有幾張布條，幾句懇託。

但五點左右，一個國民黨議員團隊全副裝備現身：音響、旗幟、擴音器，還刻意站在我們對面，高聲播放、蓄意干擾。

我們想問：

🔸 當大家都還能講話時，你們為什麼選擇用聲量蓋過別人的存在？
🔸 當對方只是靜靜舉牌，訴說對這塊土地的熱愛，你們為什麼擺出陣仗，像是要拚場？
🔸 在這樣的對比之下，誰才是真正在乎家鄉？誰才是在做樣子給上面看？

你們可以喊得再大聲，我們還是會站在這裡，不為鬥氣，只為真心。
因為我們相信，真正的改變，從不是靠音量堆疊出來的，而是靠人民，一顆一顆不退縮的心。
```
